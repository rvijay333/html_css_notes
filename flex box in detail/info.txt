in flex box concept;
u need to understand , two axis : main and cross axis . 
main = x axis 
cross = y axis

/* to apply flex on to the class box , i have to go to its parent , which is the 'container' class in this scenario*/

                                                  |---> flex-start : to align at the start  of the main axis
                                                  |---> flex-end : to align at the end of the main axis
to position on the main axis => justify-content --|---> center : to align at the center of the main axis 
                                                  |---> space-between : spaces evenly all the div's inside 


                                                  |---> flex-start : to align at the start  of the main axis
                                                  |---> flex-end : to align at the end of the main axis
to position on the cross axis => align-items -----|---> center : to align at the center of the main axis 
                                                  |---> ( many are mostly same , so i wont re write it here )

ok so when u wrap , contents ie; boxes move into new rows
assume 6 boxes => 3 in each row 

so , the each set(3 box up and 3 dox down) has its own set of cross(y) axis
so meaning, when we apply "align-item:" it gets applied to both the set separtely when the window remains shrunk

to over come this , even when the window remains shrunk , ie , to access both rows at once , we use => align-content

                                                  |---> 0 : wont shrink , will retain size when window resized
                           |---> flex-shrink : ---|---> 1 : will shrink , ie; adjust the shrinking size , as window is resized 
some important property ---|
                           |---> flex-grow : ---|---> 0 : wont grow with window resizing 
                                                |---> 1 : will grow with window resizing
